# Outputs Directory

This directory contains all analysis outputs from the 20295 Microeconometrics problem sets.

## Structure:

- **`figures/`**: All plots, graphs, and visualizations
  - Save in PDF format for LaTeX compatibility
  - Use descriptive filenames (e.g., `ps1_balance_check.pdf`)

- **`tables/`**: Analysis results and summary tables
  - LaTeX format (.tex) for inclusion in reports
  - Also save as Excel (.xlsx) or CSV for sharing
  - Example: `ps1_regression_results.tex`

- **`logs/`**: Analysis logs and computational outputs
  - Stata .log and .smcl files
  - R console outputs and debug information
  - Package version information

## Naming Convention:

Use the pattern: `ps{number}_{description}.{extension}`

Examples:
- `ps1_descriptive_stats.tex`
- `ps2_event_study_plot.pdf`
- `ps3_regression_output.log`

## Guidelines:

1. **Reproducibility**: All outputs should be generated by scripts, not manually created
2. **Version control**: Generally commit tables and figures to track changes over time
3. **Cleanup**: Remove temporary or draft outputs regularly
4. **Documentation**: Include brief descriptions in commit messages when adding new outputs
