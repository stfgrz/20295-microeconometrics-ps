# Problem Set 1 - Data

This directory contains the datasets used for Problem Set 1 of the 20295 Microeconometrics course.

## Files:
- `jtrain2.dta` / `jtrain2.csv`: Job training dataset (Lalonde format)
- `jtrain3.dta` / `jtrain3.csv`: Extended job training dataset

## Usage:
The Stata scripts in the parent directory will automatically load data from this folder using relative paths.
